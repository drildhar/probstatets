📊 Visualisasi Kekuatan Asosiasi Pembelian

Kovarians: Cov(X,Y) = E[XY] − E[X]·E[Y]
Korelasi: ρ = Cov(X,Y) / (σ_x · σ_y)
Rentang nilai ρ adalah -1 hingga +1. Jika ρ positif dan besar, fitur "Frequently Bought Together" atau bundling diskon akan sangat efektif meningkatkan Average Order Value (AOV).

🚀 Rekomendasi DevOps: Provisikan infrastruktur untuk menangani — req/detik guna memastikan SLA 75% terpenuhi saat Flash Sale. Setel sistem auto-scaling agar memicu penambahan node saat trafik menyentuh ambang — req/detik.
